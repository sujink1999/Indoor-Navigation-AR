Overview

Model data and motion data Contains.
Materials can we use the Albedo and Metallic and Normalmap in PBR
Since it is only model data and motion data, please make necessary script and Animator Controller by yourself when moving in game
(Please create by Animator Controller as you wish.Since the character movement script is not included, please describe it by yourself and produce it by yourself)

  • Texture

    • Character\4Hero\Models\Textures
      Texture_Chara_4Hero_Albedo.tga
      Texture_Chara_4Hero_MetallicSmooth.tga

    • Character \Hero\Animations

FBX Version

Texture Description

Texture Targa format has been recorded in a folder called Textures.
Orijinal Texture size is 2048pix.

Texture_Chara_4Hero_MetallicSmooth.tga Description
 R Channel is Metallic
 G Channel is Occlusion
 A Channnel is Smoothness

 

Motion confirmation method (sample)

Sample Animation Check
SampleScene.unity
Since Prefab has SampleAnimation.cs, you can check it

  1. Open SampleScene.unity

  2. When you play Unity and press the following key

  • Key"Arrow" Run

  • Key"a" Punch

  • Key"s" Kick

  • Key"d" Damage

  • Key"f" Dead

  • Key"Space" Jump
